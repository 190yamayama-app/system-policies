# プライバシーポリシー

バージョン: 1.0
最終更新日：2026年5月3日

山下システム開発事業所（以下「当事業所」）は、「MobileIssueCreator」（以下「本サービス」）の提供にあたり、ユーザーの個人情報・データの取り扱いについて以下のとおり定めます。

## 第1条（収集する情報）

当事業所は、本サービスの提供のために以下の情報を収集します。

**GitHub 認証情報（OAuth トークン）**
GitHub API へのアクセスに使用します。お客様のデバイス内にのみ保存し、当事業所のサーバーへは送信しません。

**Firebase 匿名 UID**
プッシュ通知等のサーバー連携機能のために自動的に発行される匿名の識別子です。氏名・メールアドレスなどの個人を特定できる情報とは紐付きません。

**FCM トークン（プッシュ通知トークン）**
GitHub Copilot エージェントによる作業完了等のプッシュ通知をお届けするために使用します。当事業所が管理する Firebase サーバーに保存されます。

**通知データ**
作業完了通知などの内容は、当事業所が管理する Firebase Firestore サーバーに保存されます。

**クラッシュ情報**
アプリが予期せず終了した場合、端末のモデル・OS バージョン・クラッシュ箇所などの技術情報が Firebase Crashlytics を通じて自動的に収集されます。この情報に個人を特定できる情報は含まれません。

## 第2条（情報の利用目的）

収集した情報は、以下の目的のために使用します。

1. GitHub API との通信およびユーザーが指示した操作の実行
2. プッシュ通知の送信
3. アプリの品質向上および不具合の修正
4. サービスの安全な運営と不正利用の防止

上記以外の目的でユーザーの情報を利用する場合は、事前に同意を求めます。

## 第3条（情報の保存と管理）

**デバイス内に保存される情報**
以下の情報はお客様のデバイス内にのみ保存され、当事業所のサーバーには送信されません。
- GitHub OAuth トークン（Android Keystore による EncryptedSharedPreferences で暗号化）
- アプリの設定情報（言語・テーマ・使用リポジトリ等）
- 操作ログ（監査ログ）

**サーバーに保存される情報**
以下の情報は当事業所が管理するサーバー（Firebase）に保存されます。
- FCM トークン
- 通知データ
- クラッシュ情報（匿名）

**セキュリティ対策**
当事業所は、収集した情報を不正アクセス・漏洩・改ざんから保護するために、技術的・組織的な安全管理措置を講じています。

## 第4条（第三者への提供）

当事業所は、以下の場合を除き、ユーザーの情報を第三者に提供しません。

1. ユーザーの同意がある場合
2. 法令に基づく場合
3. 人の生命・身体・財産の保護のために必要な場合

## 第5条（利用する外部サービス）

本サービスは以下の外部サービスを利用しています。各サービスのプライバシーポリシーもあわせてご確認ください。

**Firebase（Google LLC）**
プッシュ通知・データ保存・クラッシュ解析のために利用しています。
[Firebase プライバシーポリシー](https://firebase.google.com/support/privacy)

**GitHub（GitHub, Inc.）**
認証および GitHub API へのアクセスに利用しています。
[GitHub 一般プライバシーステートメント](https://docs.github.com/ja/site-policy/privacy-policies/github-general-privacy-statement)

## 第6条（ユーザーの権利）

ユーザーは以下の権利を有します。

1. 当事業所が保有するご自身のデータの開示請求
2. 誤った情報の訂正請求
3. サービスの利用停止またはデータの削除請求

これらの権利を行使する場合は、アプリの設定メニュー内「お問い合わせ」からご連絡ください。

## 第7条（データの保持期間）

当事業所は、利用目的を達成するために必要な期間に限りデータを保持します。サービスの利用を終了した場合またはデータ削除のご要望があった場合は、速やかに削除手続きを行います。デバイス内のデータは、アプリをアンインストールすることで削除されます。

## 第8条（プライバシーポリシーの改定）

本ポリシーは、必要に応じて改定されることがあります。重要な変更がある場合は、アプリ内でお知らせします。

## 第9条（お問い合わせ）

本ポリシーに関するお問い合わせは、アプリの設定メニュー内「お問い合わせ」からご連絡ください。
