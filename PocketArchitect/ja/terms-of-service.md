# 利用規約

バージョン: 1.0
最終更新日：2026年5月3日

山下システム開発事業所（以下「当事業所」）が提供する「PocketArchitect」（以下「本サービス」）の利用にあたり、以下の利用規約（以下「本規約」）に同意の上でご利用ください。

## 第1条（本サービスの概要）

本サービスは、GitHubと連携し、AI（GitHub Copilot 等）への作業指示の作成、Issue・Pull Request の管理、通知の受け取りなどを支援するスマートフォンアプリケーションです。

## 第2条（利用資格）

本サービスを利用できるのは、GitHub アカウントを保有する方に限ります。未成年の方が本サービスを利用する場合は、保護者の同意を得た上でご利用ください。

## 第3条（GitHub との連携）

1. 本サービスの利用には GitHub アカウントによる OAuth 認証が必要です。
2. ユーザーは、GitHub の利用規約を遵守した上で本サービスを利用するものとします。
3. ユーザーが GitHub 上で保有する権限の範囲内でのみ、本サービスを通じた操作を行うことができます。

## 第4条（AI 機能の利用）

1. 本サービスが AI（GitHub Copilot 等）への指示出しを支援する機能を提供しますが、当事業所は AI が生成したコード・文章・提案内容の正確性・安全性・適法性について一切保証しません。
2. AI が生成した成果物の利用・採用はユーザー自身の責任で行うものとします。
3. AI によって生成されたコードをリポジトリに反映した結果生じた損害について、当事業所は責任を負いません。

## 第5条（ユーザーの責任）

1. ユーザーは、本サービスを通じて行うすべての GitHub 操作（Issue の作成・削除、Pull Request の作成・マージ、コードの変更等）について、自己の責任において行うものとします。
2. 第三者が所有・管理するリポジトリで本サービスを利用する場合、ユーザーは当該リポジトリの所有者から適切な権限を取得していることを保証するものとします。
3. ユーザーは、OAuth トークンを厳重に管理し、紛失・漏洩した場合は速やかに GitHub 上でトークンを無効化するものとします。

## 第6条（禁止事項）

ユーザーは、本サービスの利用にあたり、以下の行為を行ってはなりません。

1. 法令または公序良俗に違反する行為
2. 当事業所または第三者の著作権・商標権・特許権その他の知的財産権を侵害する行為
3. 当事業所または第三者の名誉・プライバシーを侵害する行為
4. 本サービスを通じて権限のないリポジトリへのアクセスを試みる行為
5. 本サービスのリバースエンジニアリング、逆コンパイル、逆アセンブル
6. 当事業所の書面による許可なく、本サービスを商業目的で第三者に提供または再配布する行為
7. 本サービスの運営を妨害する行為
8. その他、当事業所が不適切と判断する行為

## 第7条（サービスの変更・中断・終了）

1. 当事業所は、ユーザーへの事前通知なく、本サービスの内容を変更または追加することができます。
2. 当事業所は、システムのメンテナンス・障害・その他の理由により、本サービスを一時的に中断することがあります。
3. 当事業所は、事前に通知した上で本サービスを終了することができます。ただし、緊急の場合はこの限りではありません。
4. 前各項によってユーザーに損害が生じた場合でも、当事業所は責任を負いません。

## 第8条（免責事項）

1. 当事業所は、以下の損害について責任を負いません。
   - AI の誤作動によるコードの不具合・データの損失
   - GitHub API の仕様変更・障害による本サービスの機能不全
   - ユーザーの操作ミスによるリポジトリへの意図しない変更・削除
   - ネットワーク障害・サーバー障害による通知の未着または遅延
   - ユーザーによるトークン管理の不備による情報漏洩
2. 当事業所の責任が認められる場合においても、その範囲はユーザーに生じた直接かつ通常の損害に限るものとします。

## 第9条（知的財産権）

本サービスに関する著作権その他の知的財産権は、当事業所または正当な権利者に帰属します。ユーザーは、本規約で明示的に認められた範囲を超えて、本サービスのコンテンツを複製・改変・配布することはできません。

## 第10条（規約の変更）

当事業所は、必要と判断した場合、本規約を変更することができます。変更後の規約はアプリ内に掲示した時点で効力を生じます。重要な変更がある場合は、アプリ起動時に改めて同意を求めます。

## 第11条（準拠法・裁判管轄）

本規約は日本法に準拠します。本規約に関する紛争については、当事業所の所在地を管轄する裁判所を第一審の専属的合意管轄裁判所とします。
